# HMS Immunology RNASeq Pipeline
This is the pipeline for RNA Seq analysis for labs in the HMS Immunology Department






#### The follow species are currently supported by the App
1. mouse: mm9, mm10 <br>
2. human: Hg19, Hg38 <br>
